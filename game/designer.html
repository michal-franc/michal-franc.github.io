Machine Designer build modular machines

Design Baselines: No hardpoint types — any weapon fits any slot. Build variety comes from tonnage (weight budget), slots (space per location), and soft bonuses/penalties from attachments. Arms/turrets/mounts give accuracy, recoil, arc, and melee bonuses — not hard weapon locks. Constraints: MaxTonnage = min(Locomotion carry, Core power, Stabilizer load). Weapons cost tonnage + slots. Armor costs tonnage, distributed front/rear per location.
